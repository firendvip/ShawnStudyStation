# 设置图标统一 · 测试用例（2026-07-01）

## 范围
全站「设置」图标统一成「错字练习(PinPin)」的 Feather 齿轮形状（circle r=3 + 齿轮 path，`stroke=currentColor`）。本次改：单词小测(去 fa-cog)、小学心算(滑块→齿轮)；转盘、单词速刷本已同形；其余页无设置图标。

## 环境
E-云端 = https://look3.cn （主站 iframe `#<view>` + 独立页 `/apps/...`，带 `?cb=x` 破缓存）；E-本地 = file:// 预览。
