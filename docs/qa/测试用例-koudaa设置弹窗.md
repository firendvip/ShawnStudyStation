# 测试用例 — 小学心算(koudaa)「设置」弹窗整合语速滑块

- **被测页面**：`vendor/koudaa/index.html`（主站 `https://look3.cn/#koudaa` 以 iframe 嵌入，亦可独立打开）
- **对应变更**：v1.66.0 —— 问号图标改为齿轮设置图标；语速滑块由卡片下方移入「设置」弹窗；弹窗排版为「设置 → 语速 → 分隔 → 使用说明」；删除弹窗内版本号。
- **优先级说明**：P0 = 阻断/核心功能；P1 = 重要；P2 = 次要/体验。
- **类型说明**：正向 / 边界 / 异常 / 回归 / 兼容。
