# 测试用例 · 单词速刷 · 手机版（s.look3.cn）

- 功能：单词速刷手机版（WordCards Mobile），移动端卡片式刷词工具，与桌面版 1:1 功能对齐。
- 被测源码：`vendor/CoCaWordCards/mobile/`（`index.html` / `app.js` / `style.css`），功能基准 `vendor/CoCaWordCards/webapp/`。
- 生成日期：2026-07-01

---

## 一、测试范围

覆盖手机版 12 个功能族：加载与词包、卡片展示、导航手势、发音、分组与区间、外观与显隐、自定义词包、设置面板（Bottom Sheet）、移动端专项、持久化与联动、部署与回归、异常与边界。不含桌面版专属交互（键盘为主的操作仅作兼容点覆盖）。

## 二、测试环境

| 代号 | 说明 |
|---|---|
| **E-云端手机** | 真机浏览器访问 `https://s.look3.cn`（iOS Safari / Android Chrome，含刘海屏 + 底部手势条机型），真实移动视口 |
| **E-本地** | 桌面 Chrome 缩窄到手机视口（DevTools 设备模拟，宽 375/390，viewport-fit=cover），或直接打开 `mobile/index.html` |

同源资源：词包 `/packs`、真人发音 `/audio`（uk/us）、`/api` 反代后端（埋点 `wordcards_m`）。主站与本站共用同一份词包与音频文件。

## 三、优先级定义

| 级别 | 含义 |
|---|---|
| **P0** | 核心链路：加载、刷词、发音、翻词、设置持久化。任一失败即阻断上线。 |
| **P1** | 重要功能：分组区间、口音切换、显隐开关、自定义词包、移动端适配。 |
| **P2** | 增强项：字号、重复间隔、边界文案、极端异常兜底。 |

## 四、通过标准

- **实际结果 = 预期结果**，且**控制台无 JS 报错**（`error` 级）、无未捕获异常。
- 音频类：**同一时刻只有一个声音**；音频 404 时降级不卡死。
- 移动端：无横向溢出、触控目标 ≥ 44px、安全区不被遮挡、页面不可缩放。
- 持久化：刷新后设置/进度保持；`__shared_wordpack` 正确写入。
