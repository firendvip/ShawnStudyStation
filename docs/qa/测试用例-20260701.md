| 项 | 说明 |
|---|---|
| 测试范围 | 本轮功能优化：①小学心算无级语速滑块 ②自然拼读视频播放 ③自然拼读切换静音 ④同一时间只一个声音 ⑤英读书屋高考范文合集收藏 |

## 2. 小学心算 · 无级语速滑块（KD）

前置：进入「小学心算」页（E2/E3 经主站，E4 直链 /apps/koudaa/index.html）。

| ID | 优先级 | 标题 | 前置/数据 | 步骤 | 预期结果 |
|---|---|---|---|---|---|
| TC-KD-001 | P0 | 滑块默认值 | 首次访问、清空 localStorage | 进入页面观察语速滑块 | 滑块默认显示 **1.5×**，手柄位于对应位置 |
| TC-KD-002 | P0 | 滑块在主站 iframe 内可见 | E3：look3.cn/#koudaa | 进入页面，不做任何滚动 | 卡片底部「语速」滑块**完整可见、不被裁切**（回归上次裁切缺陷） |
| TC-KD-003 | P1 | 默认听感=现状 | 默认 1.5× | 点播放，听一道题语音 | 语速与优化前一致（playbackRate=1.0 等效） |
| TC-KD-004 | P1 | 调慢到 1.0× | 拖滑块至最左 | 拖到 1.0×→点播放 | 显示「1.0×」；语音明显变慢；音调不失真（preservesPitch） |
| TC-KD-005 | P1 | 调快到 4.0× | 拖滑块至最右 | 拖到 4.0×→点播放 | 显示「4.0×」；语音明显变快；不出现明显「尖细变调」 |
| TC-KD-006 | P1 | 无级（连续）调节 | step=0.1 | 缓慢拖动滑块 | 数值按 0.1 连续变化（如 1.5→1.6→…），非跳跃档位 |
| TC-KD-007 | P1 | 拖动时实时生效 | 正在播放/连续出题 | 播放过程中拖动滑块 | 后续语音即时按新速率播放，无需重进 |
| TC-KD-008 | P1 | 速率持久化 | 设为 2.5× | 刷新页面 / 重新进入 | 滑块仍为 2.5×（localStorage `koudaa_speed` 生效） |
| TC-KD-009 | P2 | 边界值-下限 | — | 设为 1.0× | 不可再小于 1.0×；播放正常 |
| TC-KD-010 | P2 | 边界值-上限 | — | 设为 4.0× | 不可再大于 4.0×；播放正常 |
| TC-KD-011 | P2 | 作用于全部语音 | 设非默认速率 | 触发整句、逐词兜底、TTS 兜底三类播放 | 三类语音均按当前速率播放 |
| TC-KD-012 | P2 | 异常 localStorage | 手动写入非法值（如 abc / 10） | 进入页面 | 回退默认 1.5×，不报错、不卡死 |
| TC-KD-013 | P2 | Safari 兼容 | Safari | 调速播放 | preservesPitch（webkit 前缀）生效或合理降级，不崩溃 |
| TC-KD-014 | P2 | 移动端响应式 | 手机视口 | 查看/拖动滑块 | 滑块可见可拖、布局不溢出 |
