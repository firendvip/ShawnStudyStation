# 邮箱授权码获取指引

## 给你的说明

**目的**：拿到邮箱的「SMTP 授权码」，这样网站才能用你的邮箱发验证码给用户。

**推荐邮箱**：建议用 **QQ 邮箱** 或 **163 邮箱**，个人免费、无需任何资质，开通也最简单。

**怎么用**：
1. 打开网页端 Claude 插件（浏览器里的 Claude 插件）。
2. 把下面那一整段（四个反引号包起来的内容）**完整复制**，粘贴进插件对话框。
3. 跟着它一步步操作即可。

**重要提醒**：开启服务的过程中，通常需要你**用手机发一条短信**做安全验证。这一步插件代替不了，必须你本人用手机操作，插件会提示你什么时候发、发什么内容。

**拿到后怎么交给开发者**：插件最后会帮你整理出 5～6 行 `KEY=值` 格式的内容，把它们**直接发回到本对话**即可。注意：授权码等同于密码，只发给你信任的人。

---

## 复制下面这一整段，粘贴给网页端 Claude 插件

````
你好，我需要你引导我（一个不太懂技术的用户）在浏览器里一步步获取邮箱的 SMTP 授权码，并最后帮我整理出后端需要的几行配置。

请先问我一个问题：我用的是 QQ 邮箱 还是 163 邮箱？根据我的回答，按下面对应的步骤引导我。每次只让我做一步，确认我做完了再进行下一步。

== 如果我用 QQ 邮箱（smtp.qq.com）==
1. 引导我打开并登录 QQ 邮箱网页版：https://mail.qq.com 。
2. 让我点页面顶部的「设置」（齿轮图标 / 设置入口），进入「账号」（或「账户」）标签页。
3. 让我向下滚动，找到「POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV 服务」区域。
4. 找到「IMAP/SMTP服务」，让我点「开启」。
5. 按页面提示完成安全验证：通常要求用绑定的手机号「发送指定短信到指定号码」。请明确提示我——这一步需要我本人用手机发送这条短信，你无法代替；发完后让我回到页面点「我已发送」。
6. 验证成功后页面会弹出一串「授权码」。让我复制并保存好，这串就是 SMTP_PASS。如果我没记下，告诉我可以重新生成。

== 如果我用 163 邮箱（smtp.163.com）==
1. 引导我打开并登录 163 邮箱网页版：https://mail.163.com 。
2. 让我点顶部「设置」，进入「POP3/SMTP/IMAP」。
3. 让我开启「IMAP/SMTP服务」（或「SMTP服务」）。
4. 按提示设置 / 获取「客户端授权密码」（可能是扫码或短信验证）。这串「客户端授权密码」就是授权码（SMTP_PASS）。

== 最后，帮我整理出要交回去的配置 ==
拿到授权码后，请帮我把下面这几行填好（HOST 按我用的邮箱填；USER 填我的完整邮箱地址；PASS 填刚拿到的授权码），并把填好的整段发给我，方便我复制：

```
SMTP_HOST=smtp.qq.com        # 163 邮箱则填 smtp.163.com
SMTP_PORT=465
SMTP_SECURE=true
SMTP_USER=你的完整邮箱地址
SMTP_PASS=刚拿到的授权码
SMTP_FROM=小善学习站 <你的完整邮箱地址>
```

请注意并提醒我：
- SMTP_FROM 里的邮箱必须和 SMTP_USER 是同一个邮箱。
- SMTP_PORT 和 SMTP_SECURE 固定填 465 / true 即可，不用改。
- 授权码等同于密码，妥善保存、不要公开。
- 如果我只是想先测试，可以先用一个不重要的邮箱来试。
````
